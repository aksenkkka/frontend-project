# Frontend Project

A React frontend project built with a component-based structure and standard client-side tooling. The project includes a reusable `client` application with source files, public assets, and package configuration.

## Tech Stack

- React
- JavaScript
- CSS
- HTML
- npm


## Getting Started
Clone the repository:
git clone https://github.com/aksenkkka/frontend-project.git
cd frontend-project/client

Install dependencies:
npm install
Run the project:
npm start
The app will be available at:
http://localhost:3000
Available Scripts
Inside the client folder, you can run:
npm start
Runs the app in development mode.
npm test
Runs the test watcher.
npm run build
Builds the app for production.
